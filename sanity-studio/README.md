## Sanity Studio (AK Blog)

This is the Sanity Studio for the blog content used by the Next.js site in `ak/`.

### Run locally

```bash
cd ak/sanity-studio
npm install
npm run dev
```

### Deploy (Sanity hosting)

```bash
cd ak/sanity-studio
npm run deploy
```

### Notes

- Project: `j855mcyw`
- Dataset: `production`
- Schema type for blog posts: `post`
